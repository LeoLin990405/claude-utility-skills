# Claude Utility Skills

[English](#english) | [中文](#中文)

## English

Utility and specialized domain skills for Claude Code - skill creation, MCP building, documentation, testing, and more.

### Overview

This skill collection provides utility tools and specialized domain knowledge for various tasks.

### Skills Included

| Skill | Description |
|-------|-------------|
| `skill-creator` | Guide for creating Claude Code skills |
| `mcp-builder` | MCP server creation guide |
| `anthropic-docs` | Anthropic documentation knowledge base |
| `history-note-processor` | History note processing with Four-Step Deep Reading |
| `internal-comms` | Internal communications templates |
| `webapp-testing` | Web application testing |

### Installation

```bash
cd ~/.claude/skills
git clone https://github.com/LeoLin990405/claude-utility-skills.git
```

### Usage

```
/skill-creator           # Create new skills
/mcp-builder             # Build MCP servers
/history-note-processor  # Process history notes
/webapp-testing          # Test web applications
```

### Features

- **Skill Creator**: Templates, best practices, SKILL.md format
- **MCP Builder**: Server setup, tool definitions, resource handling
- **Anthropic Docs**: API reference, SDK usage, best practices
- **History Notes**: Four-step methodology, YAML frontmatter, Mermaid diagrams
